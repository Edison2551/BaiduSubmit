# BaiduSitemap
A typecho plugin

Based on http://zhanzhang.baidu.com/dataplug/index

License: GPLv2

## Require

php-curl
mysql

## HOW TO USE

将文件夹BaiduSubmit拷贝至`usr/plugins/`下，在后台开启即可

注:升级的时候请先禁用
